# Calculadora de Derivadas

Aplicación web para calcular derivadas de funciones matemáticas de forma simbólica.

## 🚀 Demo
https://moliaaaj.github.io/derivadas-web/

## ✨ Características

- Derivación simbólica en tiempo real
- Renderizado matemático con LaTeX
- Interfaz moderna y minimalista

## 🛠️ Tecnologías

- HTML5
- CSS3
- JavaScript
- Nerdamer
- MathJax

## 📦 Uso

Ingresar una función en términos de `x`.

Ejemplos:
- x^2 + 3*x
- log(x)
- sin(x)
- exp(x)
